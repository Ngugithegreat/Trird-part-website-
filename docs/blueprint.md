# **App Name**: TradeDesk Apex

## Core Features:

- Intelligent Signal Analysis Tool: An AI-powered reasoning engine that evaluates real-time WebSocket tick data and price patterns to decide confidence levels for upcoming market moves.
- Live Candlestick Visualization: Dynamic interactive charting utilizing lightweight-charts for deep technical analysis on various volatility indices.
- Secure OAuth Connectivity: Direct integration with the Deriv ecosystem via WebSocket API and authenticated OAuth flow for seamless account access.
- Real-time Order Execution: Advanced trading interface for Rise/Fall and Accumulator contracts with instant proposal pricing feedback.
- Historical Performance Sync: Persist trade history and personalized layout configurations using Firestore for across-session continuity.
- High-Velocity Tick Feed: Low-latency streaming tick tables providing immediate visualization of price directionality and market speed.
- Dynamic Market Ticker: Global live price marquee integrated into the application navigation for persistent cross-symbol awareness.

## Style Guidelines:

- Primary color: High-performance Emerald (#00E676) to represent growth and successful trades.
- Background color: Deep Onyx Green (#090C0B), providing a cinematic dark aesthetic with a subtle organic hue.
- Accent color: Vibrant Electric Lime (#57FF40) for highlighted action states and signal indicators.
- Primary font: 'Inter' (sans-serif) for an objective, technical, and machined aesthetic. Note: currently only Google Fonts are supported.
- Data font: 'Source Code Pro' (monospace) for displaying financial figures, ticks, and currency balances.
- Professional split-pane dashboard with a collapsible sidebar and sticky glassmorphic navigation header.
- Subtle green pulsing on the price ticker and micro-transitions for signal confidence bars to signify live data updates.
